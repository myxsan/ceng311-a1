GUARANTEED Recurrent Neural Networks (RNN)

RNNs handle sequential/variable-length inputs by feeding the previous hidden state back as input. An RNN unrolled in time = multiple copies of the same network, each passing h^t to the next step.

Pre-activation: a^t = W_h * h^(t-1) + W_i * x^t Hidden state: h^t = g(a^t) [g = tanh or sigmoid] Output: y^t = W_o * h^t Total loss: L = SUM_t L^t

BPTT (Backpropagation Through Time): gradients flow backward through all time steps via chain rule.

ParameterGradient Formula
Output weights W_o∂L/∂W_o = SUM_t delta_t_o * h_t^T
Recurrent weights W_hh∂L/∂W_hh = SUM_t delta_t_a * h_(t-1)^T
Input weights W_xh∂L/∂W_xh = SUM_t delta_t_a * x_t^T
Bias b_h∂L/∂b_h = SUM_t delta_t_a

Vanishing gradient: products < 1 repeatedly → gradients → 0 → cannot learn long-term dependencies.
Exploding gradient: products > 1 repeatedly → loss blows up → training collapses.

Solutions: gradient clipping | LSTM/GRU | truncated BPTT | better initialization

CALC wx + bias Formula & Matrix Multiplication

Single neuron: compute weighted sum, add bias, apply activation:

z = w1*x1 + w2*x2 + ... + wn*xn + b = W * x + b
y = f(z) (apply activation function)

Layer in matrix form: Z = W * X + b, then Y = f(Z) element-wise

W = [[w11, w12], x = [[x1], b = [[b1], [w21, w22]] [x2]] [b2]] Z = W * x + b = [[w11*x1 + w12*x2 + b1], [w21*x1 + w22*x2 + b2]] Then apply activation: Y = f(Z)

FunctionFormulaDerivativeNotes
Threshold
(Perceptron)		 y=1 if sum(wi*xi) > theta, else 0 Not differentiable Classification only. Cannot use in backprop.
Sigmoid
(Logistic)		 f(z) = 1 / (1 + e^(-z))
Output: (0, 1)		 f'(z) = f(z)*(1-f(z)) S-shaped, differentiable. Standard for MLP/backprop. Saturation at extremes → vanishing gradient.
Linear
(ADALINE)		 f(z) = z
Output: (-inf, +inf)		 f'(z) = 1 Gradient never vanishes. Used in ADALINE. For regression/filtering.
tanh f(z) = (e^z - e^-z) / (e^z + e^-z)
Output: (-1, 1)		 f'(z) = 1 - f(z)^2 Zero-centered. Often better than sigmoid in practice.

HOT LMS Algorithm (Least Mean Square)

Goal: minimize MSE = (1/N) * SUM(d - y)^2. Uses approximate steepest descent — estimates gradient from a single sample at a time.

Error: e(n) = d(n) - y(n) Weight: w_j(n+1) = w_j(n) + eta * e(n) * x_j(n) Bias: b(n+1) = b(n) + eta * e(n)

Stable learning requires: eta < 1 / lambda_max (largest eigenvalue of X^T * X).

Limitation: linear network can only learn linear input-output mappings.

Three cases for perceptron learning rule
ConditionError eDelta_wMeaning
Output correct (y = d)00No change
y = 0, d = 1+1+xPull weight vector toward input
y = 1, d = 0-1-xPush weight vector away from input
Unified rule: Delta_w = (d - y) * x = e * x
HOT Backpropagation Algorithm
Forward Pass (weights frozen)
For each layer l: z_l = W_l * y_(l-1) + b_l y_l = f(z_l) Compute output error: e_k = d_k - y_k
Backward Pass
Output node: delta_k = e_k * f'(v_k) Hidden node: delta_j = f'(v_j) * SUM_k(delta_k * w_kj) Updates: Delta_w_ji = eta * delta_j * y_i Delta_b_j = eta * delta_j

Energy (cost function) per sample: E = 0.5 * SUM_j (e_j)^2 — the quantity minimized by gradient descent.

Full algorithm steps: (1) Initialize weights small random → (2) Present sample → (3) Forward pass → (4) Compute output errors → (5) Backward pass with recursive delta computation → (6) Update all w and b → (7) Repeat until stopping criterion.

KEY Network Topologies & Adjacency (Weight) Matrix
TypeKey Property
Fully connectedEvery node to all others. Asymmetric connections possible. Seldom used (too many parameters).
Layered networkNodes in layers. No connection from layer j to k if j > k. Intra-layer connections may exist.
Acyclic networkNo cycles at all. No intra-layer connections in input and output layers.
FeedforwardSubclass of acyclic. Connections ONLY from layer i to layer i+1. Simplest computation.
RecurrentHas directed cycles. Previous output feeds back. More biologically realistic.
ModularSeveral sub-network modules combined for complex tasks.
Adjacency / Weight Matrix

W[i][j] = weight from node i to node j. Zero = no connection. Upper triangular = feedforward/acyclic.

Example network with nodes I, II, III, IV: I II III IV I [ 0, 0, 1.9, 0.1 ] II [ 0.4, 0, 0.3, 0.7 ] III [ 0, 0, 0, 0.8 ] IV [ 0, 0, 0, 0 ] Row = "from", Column = "to". Non-zero above diagonal only -> feedforward/acyclic network. To compute output: Z = W * x + b, then apply activation.
KEY Learning Rate Effects
Update rule: w ← w - eta * (∂E/∂w)

The gradient gives direction and relative magnitude only. eta controls the actual step size. This applies to ALL gradient-based optimizers.

HOT Overtraining, Early Stopping & Noise Injection (Week 4 last slides)
ProblemSymptomFix
UnderfittingHigh train loss AND high test loss. val_loss ≈ train_loss (both high).Add more layers/neurons, train longer
OverfittingLow train loss, HIGH test loss. val_loss >> train_loss.Early stopping, smaller network, noise injection
OvertrainingLarge network memorizes training set perfectly. Fails on test/new data.Stop before test error begins to rise
Early Stopping
  • Train weights on training set ONLY
  • Monitor test error after each epoch
  • Stop when test error starts rising
  • Confirm the rise over several iterations (avoid noise)
  • Weights never directly trained on test set → generalizes to future data
Smaller Networks & Noise Injection
  • Smaller networks: fewer parameters = less capacity to memorize = forced generalization
  • Noise injection: x_noisy = x + epsilon (small random displacement)
  • Artificially enlarges the training set
  • Prevents exact memorization of data points
  • Most effective when training data is scarce
Training curve reading: val_loss >> train_loss = overfitting  |  val_loss ≈ train_loss (both high) = underfitting  |  val_loss < train_loss = check your data split!

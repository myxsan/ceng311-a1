WEEK 1

Introduction to Neural Networks & Perceptrons

Motivation & Biological Basis

Neural networks are inspired by biological neurons — units that receive signals, process them, and fire outputs. Power comes from combining many units into large networks.

Historical milestones: McCulloch & Pitts (1943), Rosenblatt perceptron (1958), Minsky & Papert XOR critique (1969), modern deep learning revival.

Perceptron — The Simplest Unit

Binary output (0 or 1), threshold activation. Decision boundary is a line:

w1·x1 + w2·x2 + b = 0

Perceptrons can compute AND and OR — but NOT XOR, because XOR is not linearly separable (Minsky & Papert, 1969).

Solution: multi-layer networks. Hidden layers learn new feature representations that transform the problem into a linearly separable one in a new space.

Perceptron Learning Rule

Error: e = d - y. Three cases: correct (no change), y=0 d=1 (add x to w), y=1 d=0 (subtract x from w).

Delta_w = e * x     Delta_b = e
ADALINE & LMS (Delta Rule)

ADALINE uses a linear activation f(z)=z instead of a threshold. LMS minimizes mean squared error via gradient descent:

Error: e(n) = d(n) - y(n) Weight: w_j(n+1) = w_j(n) + eta * e(n) * x_j(n) Bias: b(n+1) = b(n) + eta * e(n)

Both Perceptron and ADALINE can only solve linearly separable problems. ADALINE suits regression; Perceptron suits classification.

Backpropagation (MLP Training)

Multilayer perceptrons require nonlinear, differentiable activation functions (e.g. sigmoid). Without nonlinearity, capacity reduces to a single-layer network.

Two passes:

  • Forward pass: propagate input to output with frozen weights; compute error signal e_k = d_k - y_k
  • Backward pass: propagate error gradients from output to input via chain rule; update weights
Output node gradient: delta_k = e_k * f'(v_k) Hidden node gradient: delta_j = f'(v_j) * SUM_k(delta_k * w_kj) Weight update: Delta_w_ji = eta * delta_j * y_i Bias update: Delta_b_j = eta * delta_j

Algorithm steps: (1) Initialize weights small-random → (2) Present sample → (3) Forward pass → (4) Compute errors → (5) Backward pass with recursive delta → (6) Update all w and b → (7) Repeat until stopping criterion.

WEEK 4

Training Strategies, Evaluation & Applications

Weight Initialization

Large weights cause saturation: sigmoid/tanh clips near extremes → gradients ≈ 0 → vanishing gradient → slow/no learning.

Xavier/Glorot: best for tanh/sigmoid. He/MSRA: best for ReLU. Always match init scheme to activation function.

Learning Rate η & Gradient Descent
w ← w - eta · (∂E/∂w)

The gradient gives direction and relative magnitude — eta controls the actual step size. This applies to all gradient-based optimizers (SGD, Adam, RMSProp).

Overfitting, Underfitting & Overtraining

Underfitting: model too simple. High training AND test error. Overfitting: memorizes noise. Low training error, high test error. Overtraining: large network trained too long — perfect on training set, fails on test.

Solutions:

  • Early stopping: monitor test error each epoch; stop when it starts rising (confirm over several iterations to avoid noise)
  • Smaller networks: fewer parameters = forced generalization
  • Noise injection: add small random displacement epsilon to training points; most useful when data is scarce

Training curve reading: val_loss >> train_loss = overfitting | val_loss ≈ train_loss (both high) = underfitting

Classification Metrics

Confusion matrix entries: TP, TN, FP (Type I error), FN (Type II error).

Accuracy = (TP + TN) / total Precision = TP / (TP + FP) Recall = TP / (TP + FN) F1 = 2 * Precision * Recall / (Precision + Recall) Specificity = TN / (TN + FP)

WEEK 5

Self-Organizing Maps & Convolutional Neural Networks

Unsupervised Learning & Competitive Networks

Competitive learning: neurons compete — only the winner (Best Matching Unit, BMU) updates. Losers unchanged. Winner-takes-all (WTA) strategy.

Winner: i*(x) = argmin_i || x - w_i ||^2 Weight update: Delta_w_i* = eta * (x - w_i*) Losers: Delta_w_i = 0 for i != i*

Vector Quantization (VQ): partitions input space R^n into K Voronoi regions. Minimize distortion E = SUM ||x - w_{i*(x)}||^2. Optimal: each prototype = centroid of its Voronoi region. K-Means is batch-learning VQ.

Self-Organizing Maps (SOM) — Kohonen, 1982

Maps high-dimensional data onto a 2D grid while preserving topological relationships. Each neuron has grid position r_i in R^2 AND weight vector w_i in R^n.

BMU selection: i*(x) = argmin_i || x - w_i || Weight update: w_i(t+1) = w_i(t) + eta(t) * h(i,i*,t) * (x - w_i(t)) Neighborhood (Gaussian): h(i,i*,t) = exp( -||r_i - r_i*||^2 / (2*sigma(t)^2) ) Radius decay: sigma(t) = sigma_0 * exp(-t / tau)

Two-phase training: Ordering (large eta, large sigma — global organization) → Convergence (small eta, small sigma — fine-tuning). PCA initialization gives fastest convergence.

Quality metrics: Quantization Error (QE) = prototype accuracy | Topographic Error (TE) = topology preservation.

Convolutional Neural Networks (CNN)

Designed for spatial data (images). Key components: convolutional layers (local receptive fields, shared weights), pooling layers (spatial downsampling), fully connected layers (classification).

Key property: translation invariance via weight sharing. Hierarchical feature extraction: edges → textures → objects. Huge parameter reduction vs. fully connected.

WEEK 6

Recurrent Neural Networks (RNN) & BPTT

Why RNNs?

Not all problems have fixed-length inputs/outputs. Sequential data (text, speech, time-series) requires memory of past inputs. RNNs use the previous hidden state as part of the current input. An RNN unrolled in time = multiple copies of the same network, each passing h^t to the next step.

RNN Equations
Pre-activation: a^t = W_h * h^(t-1) + W_i * x^t Hidden state: h^t = g(a^t) [g = tanh or sigmoid] Output: y^t = W_o * h^t Total loss: L = SUM_t L^t
BPTT — Backpropagation Through Time

Gradients flow backward through all time steps via chain rule. All weight matrices share parameters across time steps.

ParameterGradient
Output weights W_o∂L/∂W_o = SUM_t delta_t_o * h_t^T
Recurrent weights W_hh∂L/∂W_hh = SUM_t delta_t_a * h_(t-1)^T
Input weights W_xh∂L/∂W_xh = SUM_t delta_t_a * x_t^T
Bias b_h∂L/∂b_h = SUM_t delta_t_a

Truncated BPTT: for very long sequences, back-propagate only the last K steps to reduce computation.

Vanishing & Exploding Gradients

Vanishing: repeated multiplication of values < 1 → gradients → 0 → cannot learn long-term dependencies.

Exploding: repeated multiplication of values > 1 → gradients blow up → training instability.

Solutions: gradient clipping (cap gradient norm), LSTM/GRU architectures, truncated BPTT, better initialization.

Training Loop
For each mini-batch: 1. Initialize h_0 = 0 2. Forward pass: compute h_t and y_hat_t at each step t 3. Compute total loss L = SUM_t L^t 4. BPTT: propagate gradients backward through all time steps 5. Gradient clipping (especially for exploding gradient) 6. Optimizer step: SGD / Adam / RMSProp
